# .agent

## tasks/

- PRD를 저장
  - 기능을 구현하기 전에 계획 모드로 구현 계획 생성을 요청
  - 계획을 tasks/ 폴더에 저장

이유:

1. 디버깅 용이: 어차피 한 번에 구현 안됨. 최악의 경우 여러 세션에 걸쳐서 구현되게됨 -> 하나의 기반 문서 필요
2. 반복 작업: 비슷한 것을 구현할 때 참조할 수 있음

## system/

- 코드베이스의 중요하고 복잡한 부분
  - 프로젝트 구조
  - 데이터베이스
  - API
- 여러 개별 PRD 속해있는 경우 많음

## SOP/ (Standard Operating Procedure)

- 프로세스(지금까지의 작업/작업을 수행한 방법) 기록
- 실수 기록
