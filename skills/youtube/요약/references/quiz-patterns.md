# Quiz Patterns

3단계 퀴즈 출제 가이드.

## 출제 원칙

- **구체적 숫자/데이터 암기 문제는 절대 출제하지 않는다** (해결률 몇%, 매출 몇억, 몇 건 등)
- 거시적 관점에서 "왜", "어떤 구조로", "무엇이 본질인가"를 묻는다
- 영상의 논리 흐름과 인과관계 이해를 검증한다
- 다른 맥락에 적용할 수 있는 사고력을 테스트한다

## 난이도별 문제 유형

### 1단계 (기본) - 핵심 논지 파악

영상이 전달하려는 큰 그림과 핵심 주장:
- "발표자가 이 영상을 통해 궁극적으로 주장하는 바는?"
- "영상에서 제기한 근본적인 문제는 무엇인가?"
- "이 영상의 전체 논리 구조를 관통하는 키워드는?"

### 2단계 (중급) - 논리 구조와 인과관계

개념 간의 관계, 원인-결과, 전략의 이유:
- "X라는 접근법을 취한 근본적인 이유는?"
- "A가 실패하고 B가 성공한 구조적 차이는?"
- "발표자가 이 순서로 논지를 전개한 이유는?"

### 3단계 (심화) - 적용과 비판적 사고

다른 상황에의 적용, 한계점 인식, 전략적 함의:
- "이 접근법이 다른 산업에 적용될 때 가장 큰 장벽은?"
- "발표자의 주장에서 암묵적으로 전제하고 있는 것은?"
- "이 전략의 장기적 리스크로 추론할 수 있는 것은?"

## AskUserQuestion 형식

```
AskUserQuestion:
questions:
  - question: "[1단계 - 기본] 질문 1..."
    header: "Q1"
    options:
      - label: "A"
        description: "선택지 A 설명"
      - label: "B"
        description: "선택지 B 설명"
      - label: "C"
        description: "선택지 C 설명"
      - label: "D"
        description: "선택지 D 설명"
    multiSelect: false
  - question: "[1단계 - 기본] 질문 2..."
    header: "Q2"
    ...
  - question: "[1단계 - 기본] 질문 3..."
    header: "Q3"
    ...
```

## 결과 처리

각 단계 완료 후:
1. 정답/오답 즉시 표시
2. 틀린 문제에 대해 상세 해설 제공:
   - 정답이 무엇인지
   - 왜 그것이 정답인지 (영상 내용 참조)
   - 관련 타임스탬프 (있을 경우)
